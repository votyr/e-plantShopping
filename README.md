# 🌿 e-plantShopping (Paradise Nursery)

This project is a React-based e-commerce web application for purchasing houseplants online. It allows users to browse plants, add them to a cart, and manage their shopping experience.

## 🚀 Features

- 🌱 Browse plants categorized into:
  - Air Purifying Plants
  - Aromatic Plants
  - Medicinal Plants
- 🛒 Add plants to cart
- 🔒 Disable "Add to Cart" button after adding
- 🔢 Dynamic cart item count
- ➕ Increase / ➖ Decrease quantity
- ❌ Remove items from cart
- 💰 View total cart amount
- 🔄 Continue shopping option
- 🧾 Checkout button (Coming Soon)

## 🧠 Technologies Used

- React.js
- Redux Toolkit
- CSS
- Vite

## 🏪 About the Company

Paradise Nursery is a plant shop dedicated to bringing greenery into homes. It offers a variety of plants that improve air quality, enhance aesthetics, and promote a healthy lifestyle.

## 📁 Project Structure

- `App.jsx` – Landing page and navigation
- `ProductList.jsx` – Displays plant categories and products
- `CartItem.jsx` – Shopping cart functionality
- `CartSlice.jsx` – Redux state management
- `AboutUs.jsx` – Company details
- `App.css` – Styling and layout
